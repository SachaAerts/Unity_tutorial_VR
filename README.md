# VR Cube Manipulation – Projet d'apprentissage Unity VR

## À propos du projet
Ce projet a été développé dans le cadre de mon stage en développement VR.  
L'objectif principal était de découvrir et maîtriser les fondamentaux du développement d'applications en réalité virtuelle avec Unity, en mettant l'accent sur les systèmes d'interaction et la manipulation d'objets en VR.

## Description
Application de réalité virtuelle permettant de manipuler un cube géant dans l'espace 3D.  
Le projet met en œuvre différents types d'interactions VR pour offrir une expérience intuitive et immersive.

## Fonctionnalités

L'utilisateur peut interagir avec le cube de trois manières différentes :

### 1. Déplacement

**Avec les mains :**
- Effectuer un *pinch* (rapprocher le pouce et l'index) avec la main gauche pour sélectionner et déplacer le cube dans l'espace.

**Avec les contrôleurs :**
- Utiliser les boutons **Grip** du contrôleur gauche.

---

### 2. Rotation

**Avec les mains :**
- Sélectionner le cube avec un *pinch* de la main gauche.
- Effectuer simultanément un *pinch* avec la main droite.
- Tourner la main droite pour faire pivoter le cube selon la même rotation.

**Avec les contrôleurs :**
- Utiliser le contrôleur gauche pour effectuer la rotation.

---

### 3. Zoom / Dézoom

**Avec les mains :**
- Effectuer un *pinch* avec les deux mains simultanément tout en ayant sélectionné le cube.
- Rapprocher les mains l'une de l'autre pour zoomer.
- Éloigner les mains l'une de l'autre pour dézoomer.

**Avec les contrôleurs :**
- Utiliser le joystick droit pour zoomer ou dézoomer.

---

## Technologies utilisées

- Unity (version du projet)
- XR Interaction Toolkit
- Hand Tracking
- Support des contrôleurs VR

---

## Installation et exécution

1. Cloner le repository  
2. Ouvrir le projet avec Unity  
3. Connecter votre casque VR  
4. Lancer la scène principale  
5. Profiter de l'expérience !
